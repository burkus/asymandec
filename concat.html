Diffie-Hellman Key Exchange (1976)

Securely exchange key over an untrusted channel for the first time ever

No pre-shared keys


keysize = 2 ** 3072
g = randrange(keysize) # publicly known
p = someprime(keysize) # publicly known
a = randrange(keysize) # known only to Alice
b = randrange(keysize) # known only to Bob
A = g ** a % p # Alice's public key
B = g ** b % p # Bob's public key
shared_secret = A ** b % p

RSA (1977)

New trap door function

Moved on from Discrete Logarithm Problem (DLP)

Now focused on intractability of factoring


p = someprime(keysize / 2)
q = someprime(keysize / 2)
n = p * q # composite modulus
phi = (p - 1) * (q - 1) # phi of a prime p is p - 1
public_key = 65537 # prime numbers are always invertible
private_key = inverse(public_key, phi) # modular inverse
encryption = lambda m: pow(m, public_key, n)
decryption = lambda c: pow(c, private_key, n)

Elgamal (1985)

Asymmetric, but much closer to Diffie-Hellman than RSA

Trap door is DHP


p = 809 # public
q = 101 # public
g = randrange(1, p) # public
x = randrange(1, p) # private key
y = pow(g, x, p) # public key
def encrypt(m):
    k = randrange(1, q)
    c1 = pow(g, k, p)
    c2 = m * pow(y, k, p)
    return (c1, c2)
def decrypt(ct):
    c1, c2 = ct
    return c2 // pow(c1, x, p)
