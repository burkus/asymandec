p = someprime(2 ** 2048)
q = someprime(2 ** 2048)
n = p * q # composite modulus
phi = (p - 1) * (q - 1) # phi of a prime p is p - 1
public_key = 65537 # small prime for fast encryption
private_key = inverse(public_key, phi) # modular inverse
encryption = lambda m: pow(m, public_key, n)
decryption = lambda c: pow(c, private_key, n)

Mathematical Groups

Collection of numbers upon which a binary operation is defined

Operation must satisfy:

  • Group must be closed under it
  • Associativity
  • Identity element (0)
  • Every element has an inverse

Elgamal (1985)

Asymmetric, but much closer to Diffie-Hellman than RSA

Trap door is DHP

Keys are ephemeral

Uses a subgroup q of the prime field p

Reusable


p = 809 # prime field
q = 101 # subgroup
h = 2
g = pow(h, (p - 1) // q, p) # generator of q, has order q
x = randrange(1, q)
y = pow(g, x, p)
def enc(m):
    k = randrange(2, q)
    c1 = pow(g, k, p)
    c2 = m * pow(y, k, p)
    return (c1, c2)
def dec(ct):
    c1, c2 = ct
    return c2 // pow(c1, x, p)
