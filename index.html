RSA (1977)

New trap door function

Not using DLP

Now focused on intractability of factoring


p = someprime(2 ** 2048)
q = someprime(2 ** 2048)
n = p * q # composite modulus
phi = (p - 1) * (q - 1) # phi of a prime p is p - 1
public_key = 65537 # small prime for fast encryption
private_key = inverse(public_key, phi) # modular inverse
encryption = lambda m: pow(m, public_key, n)
decryption = lambda c: pow(c, private_key, n)

Elgamal (1985)

Extention of diffie-hellman

Relies on DHP

Uses a key tuple(x, k) where k is ephemeral

Uses a subgroup q of the prime field p


p = 809
q = 101
g = randrange(2, q)
x = randrange(1, q)
y = pow(g, x, p)
def enc(m):
    k = randrange(2, q)
    c1 = pow(g, k, p)
    c2 = m * pow(y, k, p)
    return (c1, c2)
def dec(ct):
    c1, c2 = ct
    return c2 // pow(c1, x, p)

ECElgamal


curve = curves["secp256k1"] # public
Point = curve.g # public
order = curve.n # public
x = randrange(1, order) # private key
Q = x * Point # public key
def enc(m):
    M = translateToCurve(m, curve) # place message on curve
    k = randrange(1, order) # private ephemeral key
    C1 = k * Point # public ephemeral key
    C2 = M + k * Q # encypted message
    return (C1, C2) # ciphertext pairing
def dec(ct):
    C1, C2 = ct
    return C2 - x * C1 # subtract keys from ciphertext to get M
